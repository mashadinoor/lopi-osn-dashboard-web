# 📄 Pemberitahuan Data

## Tujuan Repositori

Repositori ini **hanya berisi source code** yang digunakan untuk membangun aplikasi **Dashboard OSN**.

Repositori ini dibuat sebagai referensi pengembangan perangkat lunak dan **tidak bertujuan untuk mendistribusikan data resmi** dari instansi mana pun.

---

## Data yang Tidak Disertakan

Repositori ini **tidak menyertakan** data apa pun yang berasal dari **Pusat Prestasi Nasional (Puspresnas)** maupun pihak ketiga lainnya, termasuk namun tidak terbatas pada:

- Data peserta
- Data sekolah
- Data guru atau pembina
- Data hasil seleksi
- Data statistik resmi
- Dokumen resmi
- File PDF
- File Microsoft Excel (`.xlsx`, `.xls`)
- File CSV (`.csv`)
- Backup database
- Dataset lainnya

Seluruh data tersebut **bukan bagian dari repositori ini**.

---

## Tanggung Jawab Pengguna

Apabila Anda menggunakan aplikasi ini, Anda bertanggung jawab untuk:

- Memperoleh data dari sumber resmi yang berwenang.
- Memastikan penggunaan data sesuai dengan ketentuan hukum dan peraturan yang berlaku.
- Menjaga keamanan serta kerahasiaan data yang digunakan.

Penulis proyek ini **tidak menyediakan**, **mendistribusikan**, maupun **memberikan hak penggunaan** atas data resmi tersebut.

---

## Kebijakan Kontribusi

Mohon **jangan** melakukan commit atau pull request yang berisi:

- Dataset resmi
- Data peserta
- Data sekolah
- Dokumen hasil seleksi
- File ekspor PDF
- File Excel
- File CSV
- Backup database
- File `.env`
- API Key
- Token
- Password
- Kredensial lainnya
- Informasi sensitif lainnya

Repositori ini **hanya menerima**:

- Source code
- Dokumentasi
- Asset antarmuka (ikon, logo, ilustrasi, dan sejenisnya) yang memiliki izin untuk didistribusikan
- File konfigurasi yang tidak mengandung informasi sensitif

---

## Penafian

Seluruh hak cipta atas data resmi tetap menjadi milik instansi atau pihak yang menerbitkannya.

Penggunaan aplikasi ini **tidak memberikan hak** kepada siapa pun untuk menyalin, mendistribusikan, atau mempublikasikan data resmi tanpa izin dari pemilik data.

Apabila terdapat data resmi atau informasi sensitif yang secara tidak sengaja terunggah ke repositori ini, harap segera laporkan melalui **Issue** atau hubungi pemilik repositori agar dapat segera dihapus.

---

## Ringkasan

> **Repository ini hanya berisi source code.**
>
> Seluruh data yang digunakan pada aplikasi harus diperoleh secara sah dari sumber resmi dan **bukan merupakan bagian dari repository ini**.
